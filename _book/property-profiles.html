# Interactive ADU Placement Tool
For an interactive experience where you can place and explore ADU designs on these properties, 
**launch the interactive Shiny dashboard** by running `shiny::runApp('app.R')` in your R console.
The dashboard includes:
- **Interactive map** with click-to-place ADU functionality
- **Multiple ADU design options** (Modal 01, Dwight, Compact Studio, Large 2BR)
- **Real-time property visualization** with satellite and street view toggles
- **Complete property analysis** with all scoring metrics and filters
